---
description: how to run the licensing platform
---

1. Start the Backend Server:
// turbo
  - cd backend
  - npm run dev (or nodemon server.js)

2. Start the Frontend Application:
// turbo
  - cd frontend
  - npm run dev

3. Admin Access:
  - Default Admin Email: admin@hudi-soft.com
  - You can register the first admin via the "Setup Profile" link on the Admin Login page (/admin/login).
